## 组件分类（文件夹）说明

`page-effects`：页面特效

`time-date`：时间日期类

`page-background`：页面背景

`graphic-image`：图形图像类

`button-effects`：按钮特效类

`text-effects`：文本特效类

`status-bar`：状态栏特效

`code-generation`：代码生成类

`navigation-menu`：导航菜单类

`page-search`：页面搜索类

`online-testing`：在线测试类

`password`：密码类

`skill`：技巧类

`comprehensive`：综合类

`game`：游戏类
